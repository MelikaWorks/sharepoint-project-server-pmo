## My Role & Responsibilities

In this project, I was responsible for the end-to-end
configuration and administration of SharePoint Server
and Project Server as a PMO system.

### Key Responsibilities
- Configuring Project Web App (PWA)
- Defining Enterprise Project Types (EPT)
- Managing Project Detail Pages (PDP)
- Setting up Enterprise Resource Pool
- Assigning cost resources and build teams
- Creating custom views, grouping, and dashboards
- Supporting PMO reporting and governance

This was a real production system used by multiple
industrial projects.
