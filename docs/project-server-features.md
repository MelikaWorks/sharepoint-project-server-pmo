# Project Server Features Used

This document outlines the key Project Server and Project Web App (PWA)
features that were configured and used as part of this implementation.

The focus is on feature usage rather than technical configuration
details.

---

## Project and Portfolio Management

- Project Center for multi-project visibility
- Portfolio-level views for comparing project progress
- Project-level dashboards and summaries
- Project Detail Pages (PDP) for structured project information

---

## Task and Schedule Management

- Task hierarchies and dependency management
- Timeline and milestone visualization
- Controlled task editing through Project Web App (PWA)
- Progress tracking using % complete and task status indicators

---

## Resource and Cost Management

- Enterprise Resource Pool configuration
- Use of cost-type resources
- Build Team functionality for project-level resource assignment
- High-level cost visibility without exposing financial details

---

## Reporting and Views

- Custom views for project, task, resource, and cost perspectives
- Grouping and formatting for improved readability
- Role-oriented views for project managers and PMO users
- Monthly and project-based report organization

---

## Governance and Control

- Enterprise Project Types (EPT) to standardize project structure
- Consistent naming and structuring conventions
- Separation between project-level and portfolio-level views
- Emphasis on governance over ad-hoc customization
